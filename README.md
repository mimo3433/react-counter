# ⚡ React Counter Stepper

A simple and modern counter built with React.  
It demonstrates state management using `useState` and a clean UI styled with modern CSS (no UI libraries).

---

## 🚀 Features

- Increment and decrement counter
- Minimal and modern UI
- Centered stepper layout
- Glassmorphism-inspired design
- Smooth button interactions

---

## 🖥️ Preview

The UI consists of:
- A minus button on the left
- A live counter display in the center
- A plus button on the right

All centered on the screen with a soft gradient background.

---

## 🧠 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS (custom modern styling)
